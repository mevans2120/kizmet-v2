Kizmet Massage Homepage Concepts

Exploring homepage redesign directions based on the new logo featuring hands cradling mountains, water, and a rising sun. The goal is to create a warm, grounded aesthetic that emphasizes real text typography while honoring the existing site structure that works well.

Three Concept Directions

Each concept maintains the existing page structure (Hero, Services, About Preview, Testimonials, CTA) but explores different visual treatments inspired by the new logo.

Warm & Familiar
1. Faithful Evolution
Keeps the existing layout structure but shifts the color palette to warm browns from the logo. The safest evolution path - familiar to returning visitors but refreshed.

Design Principles

T
Real Text, Not Images
All text is live HTML/CSS, never rendered as images. This ensures accessibility, SEO, and easy CMS editing. The logo image is used only for the mark, not the wordmark.
H
Hands = Healing
The logo's cupped hands motif can be echoed in subtle ways - rounded corners, cradling layouts, protective negative space around content.
W
Warmth Over Cool
Shifting from the current sage-forward palette to a warmer brown-forward palette that matches the logo. Sage remains as a supporting accent.
